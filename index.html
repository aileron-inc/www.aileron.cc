What we do

ぼくたちが出来る事と、取り組みたい課題

Rich web application and web frontend

HTML5, CSS, WebVR, WebRTC, PWA(Progressive Web App) 等の技術や インターネットを通したリッチなユーザー体験を提供する事について様々な経験が有ります

Co-creation

多数のコラボレーションによるコンテンツの新しい作成手段の提供をしていきます

Web Crawler and Full text search

インターネット場の様々な情報を自動的に集め整頓し検索可能な状態にする事について得意としています

Ambient

世の中の空間情報、環境情報、これからのアンビエント情報社会にむけた人々の届きれていない情報伝達を支えていきたいです

RAD (Rapid Application Development)

スタートアップのMVPやCGM、メディアサービス等の議論の土台に載せる為の「動く物」までを迅速に作りあげます

Start-up support

社会の多様化における様々な新しい声をより早く実現する為に協力していきます
